Detailed Description

Class for the MPU6050 acceleration and Gyro sensor.

file MPU6050.hpp

Class that uses the acceleration and Gyro componenents of the MPU6050 sensor The standard address of the MPU6050 is 0x68 The MPU6050 makes use of the hwlib library
